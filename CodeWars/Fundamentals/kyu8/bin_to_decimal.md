## Моё решение которое прошло все тесты:
[**Задача -> CodeWars**](https://www.codewars.com/kata/57a5c31ce298a7e6b7000334/train/csharp)

### Условие:
Заполните функцию, которая преобразует двоичное число (данное как строку) в десятичное число.


##### Алгоритм таков:
1. **Пронумеровать разряды**. Для этой цели я использовал переменную **power**, которой присвоил: 
"длину строки - 1". Например, если бы мы делали это руками, то число "1101101" мы бы пронумеровывали
так: Начали бы с конца, над цифрой писали разряд - 0, следующую цифру - 1, и так далее. Но, так как мы идём
по строке с начала, то и присвоил я переменной **power** наибольший разряд, просто что бы было удобно считать.
2. **Умножить каждую цифру на 2 в степени, равной номеру её разряда.**
3. **Сложить.**

##### Решение кодом:
```
using System;

namespace Solution
{
  public static class Program
  {
    public static int binToDec(string s)
    {
      int sum = 0;
      int power = s.Length - 1;
      
      for(int i = 0; i < s.Length; i++, power--) {
        if(s[i] == '0')
          continue;
        
        int num = Int32.Parse(s[i].ToString());
        sum += Convert.ToInt32(num * Math.Pow(2, power));
      }
      
      return sum;
    }
  }
}
```
