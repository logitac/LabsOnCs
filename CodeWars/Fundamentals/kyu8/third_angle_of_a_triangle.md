## Моё решение которое прошло все тесты:
[**Задача -> CodeWars**](https://www.codewars.com/kata/5a023c426975981341000014/train/csharp)

### Условие:
Вам даются два внутренних угла (в градусах) треугольника.

Напишите функцию, чтобы вернуть третье.

Примечание: только положительные целые числа будут проверены.


##### Алгоритм таков:
1. Из 180 вычесть сумму двух известных угла. 

Такое решение основывается на теореме о сумме углов треугольника.

##### Решение кодом:
```
using System;

public static class Kata
{
  public static int OtherAngle(int a, int b)
  {
    return 180 - (a + b);
  }
}
```
