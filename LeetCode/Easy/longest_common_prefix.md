## 14. !!! Longest Common Prefix
[**Задача -> LeetCode**](https://leetcode.com/problems/longest-common-prefix/)

### Условие:
Напишите функцию, чтобы найти самую длинную общую струну префикса среди множества строк.
Если нет общего префикса, верните пустую строку "".

**Пример 1:**

**Вход:** strs = ["flower","flow","flight"]\
**Результат:** "fl"

**Пример 2:**

**Вход:** strs = ["dog","racecar","car"]\
**Результат:** ""\
**Пояснение:** среди входных строк нет общего префикса.


### Алгоритм решения:
1. Берём первую строку как образец для сравнения
2. Проходим по каждому символу первой строки, в этом цикле берём текущий символ из первой строки
3. Сравниваем этот символ с такими же позициями в других строках
4. Проверяем два условия: 1. Не вышли ли мы за длину текущей строки. 2. Совпадает ли символ текущей строке с символом
из первой строки
5. Если условие не выполняется - возвращаем найденный префикс

### Сложность алгоритма
* **Сложность по времени: O(n<sup>2</sup>)**
* **Сложность пространства: O(1)**

##### Решение кодом:
```cs
public class Solution 
{
    public string LongestCommonPrefix(string[] strs) 
    {
        string first = strs[0];

        for(int i = 0; i < first.Length; i++) {
            char currentChar = first[i];

            for(int j = 1; j < strs.Length; j++) {
                if (i >= strs[j].Length || strs[j][i] != currentChar) {
                    return first.Substring(0, i);
                }
            }
        }
        return first;
    }
}
```
