## 20. !!! Valid Parentheses
[**Задача -> LeetCode**](https://leetcode.com/problems/valid-parentheses/)

### Условие:
Учитывая строку `s` Содержит только персонажей `'('`, `')'`, `'{'`, `'}'`, `'['` и `']'`, определить, 
является ли строка ввода действительной.

Строка ввода действительна, если:

1. Открытые скобки должны быть закрыты одним и тем же типом скобок.
2. Открытые скобки должны быть закрыты в правильном порядке.
3. Каждая закрытая скобка имеет соответствующую открытую скобку того же типа.


**Пример 1:**

**Вход:** s = "()"\
**Результат:** true

**Пример 2:**

**Вход:** s = "()[]{}"\
**Результат:** true

**Пример 3:**

**Вход:** s = "(]"\
**Результат:** false

**Пример 4:**

**Вход:** s = "([])"\
**Результат:** true

**Пример 5:**

**Вход:** s = "([)]"\
**Результат:** false


### Алгоритм решения:
1. Создаём стек, он будет хранить открывающие скобки в порядке их появления.
2. Далее проходим по каждому символу строки, если это открывающая скобка, то кладём в стек
3. Если символ закрывающая скобка, то проверяем есть ли что-то в стеке, так как не может быть закрывающая скобка
без открывающей.
4. Достаём последнюю открывающую скобку и проверяем соответствие.
5. Если наша текущая закрывающая скобка соответствует открывающей скобке, то всё хорошо, в противном случае возвращаем false
6. После цикла проверяем пуст ли стек, если нет, то есть незакрытые скобки.

### Сложность алгоритма
* **Сложность по времени: O(n)**
* **Сложность пространства: O(n)**

##### Решение кодом:
```cs
public class Solution
{
    public bool IsValid(string s)
    {   
        Stack<char> st = new Stack<char>();

        foreach(char c in s) {
            if(c == '(' || c == '{' || c == '[') {
                st.Push(c);
            } else {
                if (st.Count == 0)
                    return false;
                
                char top = st.Pop();
                if(c == ')' && top != '(')
                    return false;
                else if(c == '}' && top != '{')
                    return false;
                else if(c == ']' && top != '[')
                    return false;
            }
        }

        return st.Count == 0;
    }
}
```
